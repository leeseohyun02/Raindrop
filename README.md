# 💧Raindrop  

---

### 📆제작기간 / 인원
2023.12.21 ~ 2023.12.28  

개인 프로젝트
<br></br>
### 📌구현기능
- 게임 시작 화면
- 오브젝트 생성
- 크기별로 점수가 다른 물방울
- 체력
- 게임 끝을 알리는 판넬
- 현재 점수, 최고 점수 표시
<br></br>
### 🎮게임설명
크기가 다른 각각의 물방울들이 랜덤으로 떨어지며, 캐릭터는 물방울을 피해야한다.  

가장 큰 물방울을 피하면 +3 중간 +2 작음 +1  

체력은 3번의 기회로 다 소모할 시 게임 종료가 된다.  

스코어의 점수는 저장되고, 재시작 할 수 있다.
